# قالب تقرير نشاط مدرسي

قالب HTML تفاعلي لإنشاء تقارير الأنشطة المدرسية باللغة العربية، يدعم الطباعة المباشرة وحفظ التقرير بصيغة PDF.

## ✨ المميزات

- 📝 نموذج إدخال على يمين الشاشة مع معاينة مباشرة على اليسار
- 🏛️ هيدر رسمي يحتوي على اسم المملكة، وزارة التعليم، الإدارة، المدرسة، التاريخ، الفصل، والصف
- 🇸🇦 شعار وزارة التعليم مدمج بشكل تلقائي
- 🎯 إدخال الأهداف بسهولة (كل سطر = هدف مستقل)
- 📷 إمكانية إضافة 4 صور كشواهد مع تسمية لكل صورة
- ✅ قسم للإيجابيات
- 🖨️ زر طباعة / تصدير PDF بنقرة واحدة
- 🎨 تصميم عصري بألوان رسمية وعناوين مستديرة الزوايا

## 📁 هيكل المشروع

```
school-report/
├── index.html              ← الملف الرئيسي
├── assets/
│   └── moe-logo.png        ← شعار وزارة التعليم
└── README.md
```

## 🚀 الاستخدام

افتح `index.html` في المتصفح، عبّئ الحقول، ثم اضغط زر **🖨️ طباعة / حفظ PDF**.

### عند التصدير إلى PDF
تأكّد من تفعيل خيار **"Background graphics"** (رسوم الخلفية) في نافذة الطباعة لضمان ظهور الألوان والخلفيات بشكل صحيح.

## 📤 الرفع على المستودع

ارفع المجلد كاملاً إلى المستودع مع المحافظة على نفس الهيكل، بحيث يبقى مسار الشعار `assets/moe-logo.png` صحيحاً.

## ⚖️ ملاحظة

شعار وزارة التعليم محفوظ لوزارة التعليم بالمملكة العربية السعودية ويُستخدم في هذا القالب للأغراض الرسمية المدرسية فقط.
